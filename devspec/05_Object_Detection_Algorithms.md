
# Object Detection Algorithms

Reflectivity thresholds:

5–20 dBZ: drizzle
20–30 dBZ: light rain
30–40 dBZ: moderate rain
40–50 dBZ: heavy rain
50–60 dBZ: intense rain
60+ dBZ: severe core

Velocity regions:
negative velocity → inbound
positive velocity → outbound

Rotation detection:
strong inbound adjacent to strong outbound

Debris scoring inputs:
- high reflectivity
- rotation
- low correlation coefficient
